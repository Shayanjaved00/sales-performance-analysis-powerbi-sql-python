# sales-performance-analysis-powerbi-sql-python
End-to-end Sales Performance Analysis project using Excel, MySQL, Python (Pandas, NumPy, Matplotlib), and Power BI dashboard.


## 👤 Author

**Shayan Javed**

---

## 📌 Overview

This project presents an end-to-end Sales Performance Analysis using industry-standard tools. The objective is to transform raw sales data into meaningful insights and an interactive dashboard that supports data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products
* Compare revenue across cities
* Visualize trends using charts and dashboards

---

## 🛠️ Tools & Technologies

* **Excel** → Data cleaning & preparation
* **MySQL** → Data querying & aggregation
* **Python**:

  * Pandas → Data manipulation
  * NumPy → Numerical operations
  * Matplotlib → Data visualization
* **Power BI** → Dashboard creation & KPI tracking

---

## 📂 Dataset Description

The dataset includes the following fields:

* Order ID
* Date
* Customer
* City
* Product
* Quantity
* Price

---

## 🔄 Project Workflow

### 1. Data Cleaning (Excel)

* Removed duplicates
* Checked missing values
* Ensured correct data formats
* Created **Revenue = Quantity × Price**

---

### 2. SQL Analysis (MySQL)

* Total revenue calculation
* Revenue by city
* Product performance analysis

---

### 3. Python Analysis

* Loaded dataset using Pandas
* Created revenue column
* Grouped data for insights
* Visualized results using Matplotlib

---

### 4. Power BI Dashboard

* KPI Card → Total Revenue
* Bar Chart → Revenue by City
* Bar Chart → Revenue by Product
* Filters for interactive analysis

---

## 📊 Key Insights

* Identified highest revenue-generating cities
* Determined top-performing products
* Observed sales distribution patterns
